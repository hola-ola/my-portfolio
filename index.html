The Smartass App

Smartass is a web application built with NodeJs, ExpressJs, MongoDB (and Mongoose), HandlebarsJs and CSS for styling. Smartass allows you to set your personal goals with respect to SMART method and easily track your progress thanks to visualisation of your achievemenets. The features include: authetification, friends invitation management, profile edition, goal set-up and update.
